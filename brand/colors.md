# Prompt Labs Color Palette

## Primary Colors

### Primary Teal
- **Hex:** #0F6E56
- **RGB:** 15, 110, 86
- **Uso:** CTAs, botões, accents principais

### Secondary Teal
- **Hex:** #1D9E75
- **RGB:** 29, 158, 117
- **Uso:** Hover states, interações

### Light Teal
- **Hex:** #5DCAA5
- **RGB:** 93, 202, 165
- **Uso:** Backgrounds, cards

## Neutral Colors

### Charcoal
- **Hex:** #2C2C2A
- **RGB:** 44, 44, 26
- **Uso:** Texto, headings

### Light Gray
- **Hex:** #D3D1C7
- **RGB:** 211, 209, 199
- **Uso:** Borders, dividers

### Off-white
- **Hex:** #F1EFE8
- **RGB:** 241, 239, 232
- **Uso:** Backgrounds claros

## Combinações

- **Primary + Charcoal:** Main pairing (contrast alto)
- **Teal Gradient:** #0F6E56 → #1D9E75 → #5DCAA5
- **Dark Mode:** #2C2C2A bg + #F1EFE8 text

## Quick Reference

- Backgrounds: Off-white ou Charcoal
- Text: Charcoal (light) / Off-white (dark)
- CTAs: Primary Teal (#0F6E56) → hover: (#1D9E75)
- Borders: Light Gray (#D3D1C7)
