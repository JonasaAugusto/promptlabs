## Começando

1. **Setup automático:**
   ```bash
   bash setup.sh
   ```
   (Coloque seus PDFs/PNGs no root da pasta)

2. **Commit no GitHub:**
   ```bash
   git init
   git add .
   git commit -m "Initial: Prompt Labs v1.0"
   git remote add origin https://github.com/SEU_USUARIO/prompt-labs.git
   git branch -M main
   git push -u origin main
   ```

3. **Publicar no Gumroad:**
   - Acesse: gumroad.com/promptlabs
   - Upload dos PDFs finais
